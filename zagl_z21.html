Co znaczy skrót ID? ID = Identyfikator.
Jaki jest cel stosowania ID? Służy do zróżniowania podobnych elementów.
Jakie jest znaczenie znaku #. Służy do numerowania selektora ID.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID to id="nazwa zdefiniowanego w CSS identyfikatora" czyli: Dobrze wykonane polecenie składa się z selektora, do którego planujemy powiązać identyfikator oraz z nazwy poprzedzonej znakiem #.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. ID jest jedno, a CLASS można używać dowoli.
  2. ID powinno przypisywać się tylko raz.
